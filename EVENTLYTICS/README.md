# Eventlytics

Eventlytics is a beginner-friendly data analytics project that studies how moving house affects food spending behavior.

## Tools Used
- Python
- Pandas
- CSV
- SQL
- Power BI planned

## Project Structure

- DATA/ contains the CSV datasets
- PYTHON/ contains the analysis script
- SQL/ contains database schema and SQL queries

## What the Project Does

The project compares food spending before and after a moving event for each user.

## Key Insights

- Walter White had the highest spending increase after moving.
- Jesse Pinkman, Daniel Horcus, and Harry Potter had lower spending after moving.
- Vikram Singh had a small increase after moving.
